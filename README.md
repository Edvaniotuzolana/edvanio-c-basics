# Edvanio C Project

## Descrição
Programa inicial em C para demonstrar estrutura básica, compilação e execução.

## Objetivos

- Melhorar conhecimentos em linguagem C
- Praticar compilação e execução de programas
- Trabalhar com estruturas de dados
- Aprender conceitos de baixo nível

## Tecnologias utilizadas

- Linguagem C
- GCC / Clang
- Linux Terminal
